## 🛠️ متطلبات التشغيل (Requirements)

لضمان تشغيل الأكواد بكفاءة على جهازك الشخصي، يفضل توفر ما يلي:
* **Compiler:** GCC (MinGW) أو Visual Studio C++ Compiler.
* **IDE:** يفضل استخدام **Visual Studio 2022** أو **VS Code**.
* **Standard:** الأكواد متوافقة مع معيار **C++11** أو أحدث.
